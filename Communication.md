# 1.史密斯圆图

## 1.1 史密斯圆图介绍

史密斯圆图（Smith Chart）是一种用于射频工程和微波工程中的图形工具，用来简化和直观地处理与传输线、阻抗匹配、反射系数等相关的复杂计算。

**史密斯圆图的基本概念**

1.**阻抗匹配**：在射频系统中，阻抗匹配至关重要，目的是最大化功率传输并最小化反射。史密斯圆图能够帮助工程师在复杂的阻抗匹配过程中，直观地找到合适的匹配点。

2.**圆图的坐标系**：
- 横坐标：代表阻抗的实部（电阻成分）。
- 纵坐标：代表阻抗的虚部（电抗成分）。

3.**圆形轨迹**：史密斯圆图上每个点代表一个复数阻抗或导纳值，圆图上的不同圆代表相同反射系数的点。随着传输线长度或频率变化，阻抗或导纳在圆图上沿着特定的轨迹移动。

4.**用途**：
- **反射系数（Reflection Coefficient）**：圆图中央点表示无反射（完全匹配），离中心越远，反射系数越大。
- **归一化阻抗和导纳**：常用来处理归一化的阻抗和导纳值，能够直观地看到阻抗匹配的变化。

5.**阻抗匹配的应用**：通过史密斯圆图，可以直观地选择匹配网络元件（如电容、电感）来实现阻抗匹配，优化系统性能。

**简单总结**

史密斯圆图是射频工程中的工具，用于简化和可视化复杂的阻抗匹配和反射系数计算。它将阻抗和导纳以图形方式表示，使得工程师可以通过图形分析来解决传输线问题和优化射频系统设计。

#### 步骤：

1. **归一化阻抗**：
   - 首先将天线的阻抗归一化，以传输线的特性阻抗为基准：
     \[
     Z_{in} = \frac{50 + j25}{50} = 1 + j0.5
     \]
   - 归一化后，阻抗为 **1 + j0.5**。

2. **在史密斯圆图上标出归一化阻抗**：
   - 在史密斯圆图上找到对应的 **1 + j0.5** 的位置。它会落在实轴上方，表示该点的阻抗有电阻和正电抗（电感性）。

3. **添加匹配元件**：
   - **目标**：将 **1 + j0.5** 移动到史密斯圆图的中心（即 **1 + j0**），表示完全匹配。

     X_C = \frac{1}{2\pi f C}

## 1.3 GaN基PA与史密斯圆图的关系

### GaN基PA与史密斯圆图的关系

虽然史密斯圆图是一种通用的射频设计工具，适用于各种功率放大器（PA），但在GaN基PA的设计中，史密斯圆图的应用需要特别考虑GaN材料的独特特性。以下是GaN基PA与史密斯圆图关系的几个关键方面：

#### 1. 高功率密度
- **影响**：GaN器件能够处理更高的功率密度，这要求匹配网络具有更高的功率处理能力和良好的热管理。
- **史密斯圆图的应用**：高功率密度要求在史密斯圆图上设计的匹配网络更加精确，以避免功率损耗和器件过热。

#### 2. 高频操作
- **影响**：GaN器件常用于微波和毫米波频段，在高频下匹配网络的设计更加复杂，寄生效应显著。
- **史密斯圆图的应用**：在高频设计中，史密斯圆图上的微小偏差可能导致性能变化，因此匹配需要非常精确，可能需要多步匹配网络来优化性能。

#### 3. 宽带匹配
- **影响**：GaN PA通常用于宽带应用，这要求在多个频率点上实现阻抗匹配。
- **史密斯圆图的应用**：在史密斯圆图上，宽带匹配要求在多个频率下进行阻抗匹配，通常需要设计多段匹配网络或使用宽带匹配技术，如带通滤波器或复合匹配网络。

#### 4. 非线性特性
- **影响**：GaN器件的非线性特性会影响其输入和输出阻抗，特别是在高功率下。
- **史密斯圆图的应用**：处理非线性效应时，需要在史密斯圆图上分析器件在不同功率水平下的阻抗变化，以确保在实际工作条件下的匹配效果。

#### 5. 热管理和可靠性
- **影响**：GaN器件在高功率和高频下的工作条件可能导致显著的热效应，这会影响阻抗匹配。
- **史密斯圆图的应用**：在设计匹配网络时，必须考虑热效应对器件阻抗特性的影响，并在史密斯圆图上进行动态调整。

### 结论
史密斯圆图在GaN基PA设计中的应用不仅仅是一个阻抗匹配工具，还需要结合GaN材料的独特电特性。考虑到GaN的高功率密度、高频操作、宽带匹配和非线性特性，设计时需要在史密斯圆图上进行更精细的匹配分析，以实现最佳的PA性能。

# 2. 741运算放大器的简单例子：反相放大器电路

## 反相放大器电路

### 电路概述
反相放大器电路是一种基本的运算放大器应用，能够将输入信号进行放大，并且输出信号与输入信号相位相反。

### 电路图

           +Vcc
            |
            |
            |
           \|/
            |
          -----
         |     |
         |  741| 
         |     |
          -----
      3 |     | 6
   +--- |  2  |---+---------------------->
   |    | -   |   |       输出电压Vout
   |    |     |   |
   |    |     |   |
   R1   |  4  |   |  R2
   |    |     |   |
 Vin--->|+    |---+ 
       -      |
       |     5|
      GND    -Vcc


### 电路说明
1. **输入信号（Vin）**：将输入信号接入运算放大器的负输入端（引脚2）。此电路的输入电压是我们希望放大的信号。

2. **反馈电阻（R1 和 R2）**：
   - **R1**：接在输入信号（Vin）与运算放大器的负输入端之间。
   - **R2**：接在运算放大器的输出端（引脚6）与负输入端（引脚2）之间。

3. **输出信号（Vout）**：输出信号从运算放大器的输出端（引脚6）获取。它与输入信号Vin相反相，并被放大。

4. **电源连接**：运算放大器的正电源引脚（引脚7）连接到正电源（如+15V），负电源引脚（引脚4）连接到负电源（如-15V）。

### 增益计算
输出电压（Vout）与输入电压（Vin）的关系由下式决定：
\[ V_{out} = -\frac{R2}{R1} \times V_{in} \]

- 如果 **R2 = R1**，那么电路的电压增益就是 -1，即输出电压的大小与输入电压相等，但相位相反。
- 如果 **R2 > R1**，输出电压会被放大且相位相反。

### 例子
假设 **R1 = 10kΩ**，**R2 = 100kΩ**，输入电压 **Vin = 1V**。  
输出电压 **Vout** 就是：
\[ V_{out} = -\frac{100kΩ}{10kΩ} \times 1V = -10V \]

所以，输出电压是 **-10V**，这意味着输入电压被放大了10倍且相位反转。

这个反相放大器电路是理解运算放大器工作原理的一个非常基础且常见的例子。

RS-232 接口的通用 PIC 微控制器板是一种嵌入式开发平台，专为连接和通信而设计。它通常包含一个 PIC 微控制器（如 PIC16F、PIC18F 系列），具备处理输入输出信号、执行程序指令的能力，并支持使用 RS-232 标准进行串行通信。

**易于开发:** 通常配备适配的编程器和开发环境（如 MPLAB X IDE），使得开发者可以方便地编写、调试和烧录程序。

工业自动化: 通过 RS-232 接口与旧式设备或工业终端进行通信和控制。
家庭自动化: 连接各种家电和控制系统，实现智能家居应用。

# 4. 在单模、多芯和空芯光纤中实现长距离光纤供电和光纤模拟无线电同步传输

**摘要：**

本文提出了一项关于在长距离空心光纤 (HCF) 上共享光纤供电 (PoF) 和第五代 (5G) 新无线电 (NR) 光纤模拟无线电 (ARoF) 的可行性的新研究。对相似长度的 HCF、多芯 (MCF) 和单模光纤 (SMF) 进行了比较测量。对受激拉曼散射 (SRS) 引起的从 PoF 到 ARoF 的噪声传输进行了全面分析。对所有测试的光纤测量了 ARoF 信号的相对强度噪声 (RIN)，观察到 HCF 没有衰减，MCF 也没有衰减，SMF 也没有明显衰减。在单根光纤中实现了 PoF 和 5G NR ARoF 链路的组合，不同调制格式的 5G NR 信号成功地通过 HCF 和 MCF 传输而没有衰减，而 SMF 中的 SRS 引起的噪声传输使它们严重衰减。还分析了 PoF 传输效率，3.1 千米 HCF 的系统能效 (SEE) 高达 9.9%，11.1 千米 HCF 的系统能效 (SEE) 高达 0.9%，10 千米 MCF、9.8 千米 SMF 和 3.3 千米 SMF 的系统能效分别为 1.5%、13.5% 和 20.4%。PoF 信号用于为低功耗蓝牙 (BLE) 负载供电，功耗为数学方程20毫瓦。

**结论：**

本文的主要目的是研究在特定系统场景中可能存在的挑战，即使用光为距离发射机几公里的接收点的低功耗设备供电。我们表明，虽然传统的 SMF 不能在我们在此处研究的功率水平下使用，但基于 HCF 或 MCF 的解决方案是可行的选择。另一方面，在专用场景中，使用单个 HCF 或 MCF 比使用 SMF 进行数据传输和使用 MMF 进行电力传输更具优势。主要优势是在电缆中占用相同的体积并使用能够满足全球标准（包括尺寸标准）的光纤来提供这两种功能。在大多数地面网络段中，最好尽可能高效地利用现有的电缆管道，因为部署新的管道基础设施可能非常昂贵。寻求在更长距离上实现更高比特率的升级有利于开发和安装 MCF 和 HCF，而不是 SMF 和 MMF。将 HCF 与 MCF 进行比较，MCF 具有有限的 XT，这可能会导致来自不同能量核心的功率传输波动，而不同的损耗会提供不太稳定的能量传输，这也会影响 EVM。

更具体地说，在我们的实验中，我们研究并证明了在共享 5G NR ARoF 和 PoF 链路中 HCF 链路用于光前传的可行性。在 HCF 中，非线性效应实际上不存在。从 PoF 信号到 5G NR ARoF 信号没有 SRS 或噪声传输的痕迹。当通过 HCF 与 PoF 一起传输时，任何测试的 5G NR ARoF 信号的质量均未下降。然而，HCF 的高衰减损害了 PoF 链路的效率，并且必须在 11.1 公里 HCF 的情况下加入 EDFA。在 MCF 的情况下，获得了相同的结果，由于 PoF 和数据信号通过不同的芯线传输，因此不存在与非线性效应相关的损失。

另一方面，SMF 中的非线性效应会导致 5G NR ARoF 信号受损。SRS 引起的噪声传输严重损害了 3.3 公里和 9.8 公里 SMF 中的 5G NR 信号质量。当 PoF 功率增加时，我们测量了 LD 的 RIN均方根的恶化。同样，在测试的三个测试模型中，测量到 5G NR ARoF EVM 恶化，所有模型的值都超过了标准限值。由于衰减较低，SMF 的 PoF 效率更高。

我们还使用 PoF 提供了 BLE 电子负载。提供的功率足以为测试的两个 SMF 和 3.1 公里 HCF 提供负载。但是，11.1 公里 HCF 的衰减太高，这降低了光电转换效率，导致接收到的电能不足以为负载供电。

总之，我们已经证明，通过单根光纤（HCF 和 MCF）长距离共享传输 5G NR ARoF 和 PoF 信号都是可行的。与通过 SMF 共享传输相比，主要优势在于非线性效应发生率较低，从而能够实现更高功率的传输，而不会影响传输数据的质量。此外，对于 HCF，更高的光纤熔断极限允许可扩展性以提供更大的功率，而不会出现光纤断裂的风险。

然而，与 SMF 相比，我们特定的 11.1 千米 HCF 的衰减相对较高，因此需要使用放大功能，而 SMF 则不需要。从长远来看，在衰减方面改善 HCF 性能时（模拟预测损耗系数将低至数学方程−1，非线性系数比实心光纤低 4 个数量级[ 34 , 35 ]，传输效率将优于 SMF 和 MCF。

## 5.1 射频基础知识：天线如何将电信号转换为电磁信号并发送/接收？

天线充当换能器，将电信号转换为电磁波，反之亦然。这种转变依赖于电磁学的基本原理。

以下是分步说明：

1、电信号作为输入：
无线电发射器以载波频率（通常是交流电 （AC））产生电信号，其中包含音频、视频或数据等信息。
该交流信号以特定频率振荡并提供给天线的端子。

2. 电荷加速和磁场生成：
天线中的振荡电流导致其金属导体内的电子来回振动或振荡。
这种电子运动是加速的一种形式，根据麦克斯韦方程组，加速电荷会产生电磁场。
具体来说，振荡电流在天线周围产生时变电场和磁场。

3、电磁波的传播：
这些随时变的电场和磁场有着内在的联系，并以电磁波（也称为无线电波）的形式传播在一起。
电场和磁场彼此垂直并与传播方向垂直。
这些波以光速从天线向外辐射到自由空间。

4、关键原则：
这种转换的基本机制是带电粒子的加速。当天线导电材料内的电子因交流电而加速时，它们会辐射电磁能。

类比：
可以把它想象成一个产生声波的扬声器。扬声器将振荡的电子转化为空气中的波。同样，天线将振荡电子转换为穿过空间的电磁波。
 
从本质上讲，天线充当电路内的电信号与在空气中传播的电磁波之间的接口，从而实现无线通信。
